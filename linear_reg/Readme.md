# Salary Prediction using Linear Regression

## Objective
The main goal of this project is to understand how different factors affect salary in the dataset taken from https://www.kaggle.com/datasets/rkiattisak/salaly-prediction-for-beginer. \
The dataset includes features like age, years of experience, education level, and gender. Using these, a linear regression model is built to predict salary.

## Steps Performed
1. Loaded and explored the dataset
2. Cleaned the data by removing missing values
3. Converted data types wherever required
4. Removed unnecessary columns (like job title)
5. Encoded categorical variables (education level, gender)
6. Split the data into training and testing sets
7. Checked correlations between variables
8. Applied linear regression to predict salary

## Key Findings
1. Years of experience has a strong positive impact on salary
2. Higher education levels generally lead to higher salaries
3. Age also shows some correlation with salary
4. Gender impact is present but less significant compared to experience and education
5. The linear regression model is able to capture general trends but may not be perfect for complex patterns

## Conclusion
This project shows that salary can be reasonably predicted using basic features and a simple regression model. However, more advanced models and additional features could improve accuracy further.
